# 业余挣钱项目

在网络做副业赚钱，方法很多，平台也很多，尤其是兼职类，适合新手小白简单粗暴的赚钱方法，每天多赚一些零花钱轻轻松松。

我用了2天时间，整合了130个副业赚钱平台，希望对大家有用。

做任务类的副业比较简单，没什么难度，也适合所有人，不过收益也有限，每天能赚个几十块钱。
